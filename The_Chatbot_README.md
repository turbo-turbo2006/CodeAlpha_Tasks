# 🤖 Simple Rule-Based Chatbot

## 💬 What is this?
This is a straightforward, no-nonsense text chatbot built entirely on basic logic rules. It doesn't use complex machine learning models or require an internet connection.
Instead, it’s a lightweight script designed to demonstrate how simple condition-based programming can simulate a conversation. It's not going to write a novel for you, but it will politely reply exactly how you tell it to.

## ⚙️ How it Works
At its core, this bot uses simple `if/else` statements to figure out what to say. 
*   **Keyword Matching:** It scans the text you type for specific words or phrases.
*   **Direct Responses:** When it recognizes a trigger word, it immediately replies with a pre-written answer.
*   **The Fallback:** If you type something completely outside its vocabulary, it relies on a default catch-all response to keep things moving.

## ✨ Why Build This?
Even though the code is incredibly short, it’s a perfect foundational project for understanding how programs take user input, process it, and return an output. It’s a great starting point before moving on to more complex natural language processing. 

## 🎯 Features
*   **100% Predictable:** You always know exactly how it will react.
*   **Easily Expandable:** Teaching the bot a new response is as easy as adding a single line of code.
*   **Fully Offline:** Runs entirely on your local machine.
